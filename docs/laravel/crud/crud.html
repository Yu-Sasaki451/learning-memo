CRUD

Todoを例にしたCreate、Read、Update、Deleteの基本形

共通設定

createやupdateで代入できるカラムをモデルに書く。

app/Models/Todo.php

protected $fillable = [
    'content',
];

コントローラのuse

use App\Http\Requests\TodoRequest;
use App\Models\Todo;

Create

入力されたTodoを保存する処理。

routes/web.php

Route::post('/todos', [TodoController::class, 'store']);

コントローラ

public function store(TodoRequest $request)
{
    Todo::create($request->only(['content']));

    return redirect('/');
}

Blade

<form class="create-form" action="/todos" method="post">
    @csrf
    <input
        class="create-form__input"
        type="text"
        name="content"
        value="{{ old('content') }}"
    >
    <button class="create-form__button" type="submit">作成</button>
</form>

Read

保存済みのTodoを一覧表示する処理。

routes/web.php

Route::get('/', [TodoController::class, 'index']);

コントローラ

public function index()
{
    $todos = Todo::all();

    return view('index', compact('todos'));
}

Blade

@foreach($todos as $todo)
    <p>{{ $todo->content }}</p>
@endforeach

Update

既存のTodoを書き換える処理。PATCHは一部更新、PUTは全体更新の意味。

routes/web.php

Route::patch('/todos/{todo}', [TodoController::class, 'update']);

コントローラ

public function update(TodoRequest $request, Todo $todo)
{
    $todo->update($request->only(['content']));

    return redirect('/');
}

Blade

<form action="/todos/{{ $todo->id }}" method="post">
    @method('PATCH')
    @csrf
    <input
        type="text"
        name="content"
        value="{{ old('content', $todo->content) }}"
    >
    <button type="submit">更新</button>
</form>

Delete

既存のTodoを削除する処理。

routes/web.php

Route::delete('/todos/{todo}', [TodoController::class, 'destroy']);

コントローラ

public function destroy(Todo $todo)
{
    $todo->delete();

    return redirect('/');
}

Blade

<form action="/todos/{{ $todo->id }}" method="post">
    @method('DELETE')
    @csrf
    <button class="delete-button" type="submit">削除</button>
</form>
