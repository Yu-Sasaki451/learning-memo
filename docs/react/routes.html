React ルーティング

ここはReact側の作業。
React RouterでURLと画面の対応を作るところだけを扱う。

このページでやること

React Router公式では、BrowserRouterでアプリ全体を包み、RoutesとRouteでURLと表示する画面を対応させる。
ここでは、/ でIndex、/category でCategoryを表示するところまで作る。

React Routerをインストール

現行のReact Router公式に合わせて、react-routerを入れる。


            ./vendor/bin/sail npm install react-router

Laravel側のルーティングを変更

routes/web.phpに書く。React側で画面遷移するため、どのURLでもindex.blade.phpを返す設定にする。


            Route::get('/{any}', function () {
                return view('index');
            })->where('any', '.*');

ルーティング用フォルダを作成

URL定義、ルート定義、ページファイル、ページCSSを分ける。


            mkdir -p resources/js/routes resources/js/pages resources/css/pages

URLを定義する

resources/js/routes/routes.jsを作る。URLを1箇所にまとめると、後で変更しやすい。


            export const ROUTES = {
                HOME: '/',
                CATEGORY: '/category',
            };

Indexページを作成

resources/js/pages/Index.jsxを作る。まずはルーティング確認用の最小コードにする。


            import '../../css/pages/Index.css';

            const Index = () => {
                return (
                    <main className="indexPage">
                        <h1>Todo</h1>
                    </main>
                );
            };

            export default Index;

IndexページのCSSを作成

resources/css/pages/Index.cssを書く。


            .indexPage {
                max-width: 960px;
                margin: 0 auto;
                padding: 32px 24px;
            }

Categoryページを作成

resources/js/pages/Category.jsxを作る。


            import '../../css/pages/Category.css';

            const Category = () => {
                return (
                    <main className="categoryPage">
                        <h1>Category</h1>
                    </main>
                );
            };

            export default Category;

CategoryページのCSSを作成

resources/css/pages/Category.cssを書く。


            .categoryPage {
                max-width: 960px;
                margin: 0 auto;
                padding: 32px 24px;
            }

定義したURLを使う

resources/js/routes/AppRoutes.jsxを作る。RouteでURLと表示するページを対応させる。


            import Index from '../pages/Index';
            import Category from '../pages/Category';
            import { ROUTES } from './routes';
            import { Routes, Route } from 'react-router';

            function AppRoutes() {
                return (
                    <Routes>
                        <Route path={ROUTES.HOME} element={<Index />} />
                        <Route path={ROUTES.CATEGORY} element={<Category />} />
                    </Routes>
                );
            }

            export default AppRoutes;

app.jsxでルートを読み込む

resources/js/app.jsxを書く。BrowserRouterでAppRoutesを包む。


            import React from 'react';
            import ReactDOM from 'react-dom/client';
            import { BrowserRouter } from 'react-router';
            import AppRoutes from './routes/AppRoutes';

            ReactDOM.createRoot(document.getElementById('app')).render(
                <BrowserRouter>
                    <AppRoutes />
                </BrowserRouter>
            );

Viteを起動して確認

http://localhost と http://localhost/category の両方を確認する。


            ./vendor/bin/sail npm run dev
