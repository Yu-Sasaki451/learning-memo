# 学習メモ

Laravel、React、JavaScript、TypeScript、CSS、環境構築、コマンドラインを見返すための学習メモです。

## 公開方法

GitHub Pagesで公開します。

公開URL:

```text
https://yu-sasaki451.github.io/learning-memo/
```

リポジトリの `Settings > Pages` で次の設定にします。

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

公開後のURLは、通常この形になります。

```text
https://GitHubユーザー名.github.io/リポジトリ名/
```
