必要なページ

  • URLから処理を追う
  • PHP基礎とよく見る書き方
  • Controllerの読み方
  • Modelの書き方
  • Viewの書き方
URLから処理を追う 画面URLから Controller、action、View を特定する時に見る。
PHP基礎 array()、[]、=>、.、.=、&&、|| を読む時に見る。
Controller $this->User、$this->request->data、$this->set()、redirect() を読む時に見る。
Model find()、save()、$validate、関連、beforeSave() を読む時に見る。
View .ctp、FormHelper、HtmlHelper、Paginator、element() を読む時に見る。

コードを読む時の検索語

分からないコードを読む時は、名前で追う。
URL、Controller名、action名、Model名、View内の文言で検索する。
