自分で開発環境構築①

ルートディレクトリを作成して移動

laravel-practiceという名前で作る例。別名にしたい場合は、1行目と2行目のlaravel-practiceだけ同じ名前に変える。


            mkdir laravel-practice
            cd laravel-practice

Docker用ディレクトリを作成

docker-compose.ymlで使うパスに合わせて、dockerの中にmysql、nginx、phpを作る。-pは途中のフォルダもまとめて作るためのオプション。


            mkdir -p docker/mysql/data docker/nginx docker/php src

docker-compose.ymlを作成


            touch docker-compose.yml

MySQL設定ファイルを作成


            touch docker/mysql/my.cnf

Nginx設定ファイルを作成


            touch docker/nginx/default.conf

PHP設定ファイルを作成


            touch docker/php/php.ini docker/php/Dockerfile
