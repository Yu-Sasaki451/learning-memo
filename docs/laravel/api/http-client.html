Laravel 外部API通信

Laravelから外部サービスのAPIを呼ぶときの基本。
Httpファサードを使う。

公式で確認したこと

Laravel公式では、外部WebアプリケーションへHTTPリクエストを送るためにHttpファサードを使う。
get、post、put、patch、deleteなどのメソッドが用意されている。

  • Laravel公式 HTTP Client

GETで取得する

外部APIからデータを取る一番基本の形。
acceptJsonは、JSONで返してほしいと伝える設定。 


            use Illuminate\Support\Facades\Http;

            $response = Http::acceptJson()->get('https://example.com/api/items');

            $items = $response->json();

クエリパラメータを送る

検索条件やページ番号を送るときは、getの第2引数に配列で渡す。
例では ?keyword=todo&page=1 のようなURLになる。 


            use Illuminate\Support\Facades\Http;

            $response = Http::acceptJson()->get('https://example.com/api/items', [
                'keyword' => 'todo',
                'page' => 1,
            ]);

            $items = $response->json();

POSTで登録する

外部APIへ登録データを送るときはpostを使う。
LaravelのHttpクライアントは、配列をJSONとして送る用途に使いやすい。 


            use Illuminate\Support\Facades\Http;

            $response = Http::acceptJson()->post('https://example.com/api/items', [
                'title' => '買い物に行く',
            ]);

            $createdItem = $response->json();

Bearerトークンを付ける

APIキーやアクセストークンが必要なAPIでは、withTokenを使う。
トークンはコードに直接書かず、.envに入れる。

.env


                EXTERNAL_API_TOKEN=ここにトークンを書く

PHP


                use Illuminate\Support\Facades\Http;

                $response = Http::withToken(config('services.external.token'))
                    ->acceptJson()
                    ->get('https://example.com/api/items');

                $items = $response->json();

config/services.phpに設定する

.envの値はconfig経由で読む形にしておくと、あとで管理しやすい。
services.phpにexternalという設定を追加する。 


            'external' => [
                'token' => env('EXTERNAL_API_TOKEN'),
            ],

失敗時の処理を書く

APIは失敗することがあるので、successfulで成功したか確認する。
失敗したらログに残して、画面や呼び出し元へ分かる形で返す。 


            use Illuminate\Support\Facades\Http;
            use Illuminate\Support\Facades\Log;

            $response = Http::timeout(10)
                ->acceptJson()
                ->get('https://example.com/api/items');

            if ($response->successful()) {
                return $response->json();
            }

            Log::error('外部API通信に失敗しました', [
                'status' => $response->status(),
                'body' => $response->body(),
            ]);

            return [];
