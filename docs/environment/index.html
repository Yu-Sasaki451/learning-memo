環境構築の後にやること

環境構築は、LaravelとReactを書き始めるための準備。
起動できたら、次は全体の流れを見て、どのファイルに何を書くかを確認する。

1. Laravelを作る SailやDockerでプロジェクトを作る。
2. Reactを表示する Blade、Vite、app.jsxをつなぐ。
3. 全体の流れを見る Laravel、React、API、DBの関係を確認する。
4. 機能を作る DB、API、React画面の順に小さく作る。
