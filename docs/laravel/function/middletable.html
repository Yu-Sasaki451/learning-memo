中間テーブル

中間テーブルを作る

多対多のリレーションで使うテーブル。
基本はモデル名を単数形にして、アルファベット順でつなげた名前にする。
例：album_song、product_season

中間テーブルにカラムを追加

song_idとalbum_idを持つ中間テーブルの例。

$table->foreignId('song_id')
    ->constrained('songs')
    ->cascadeOnDelete();

$table->foreignId('album_id')
    ->constrained('albums')
    ->cascadeOnDelete();

$table->unique(['song_id', 'album_id']);
$table->timestamps();

リレーションを定義

中間テーブルとリレーションしたテーブルのモデルに定義する。
テーブル名をLaravelの規約と違う名前にした場合は、第2引数にテーブル名を書く。

Albumモデル

public function songs()
{
    return $this->belongsToMany(Song::class, 'song_album', 'album_id', 'song_id')
        ->withTimestamps();
}

Songモデル

public function albums()
{
    return $this->belongsToMany(Album::class, 'song_album', 'song_id', 'album_id')
        ->withTimestamps();
}
