ページネーション

PreviousとNextだけ表示

ページ番号が不要で、前へ・次へだけで良い場合はsimplePaginateを使う。

コントローラ

public function index()
{
    $authors = Author::simplePaginate(4);

    return view('index', ['authors' => $authors]);
}

Blade

{{ $authors->links() }}

ページ数も表示する

ページ番号も表示したい場合はpaginateを使う。

コントローラ

public function index()
{
    $authors = Author::paginate(4);

    return view('index', ['authors' => $authors]);
}

Blade

{{ $authors->links() }}

CSS

svg.w-5.h-5 {
    width: 30px;
    height: 30px;
}
