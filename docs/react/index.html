React

Reactのコードを見る前に

ReactはHTMLっぽく見えるが、JavaScriptの値や関数を混ぜて書く。
まずはタグ、props、{}、mapの形を見る。

<Header /> 部品を表示 Headerコンポーネントを画面に出す。
<Button title="登録" /> propsを渡す Buttonにtitleを渡す。
{title} {} はJSを書く場所 JSXの中でJavaScriptの値を使う。
{todos.map(...)} 配列を一覧表示 todos配列を1件ずつ画面に出す。

Reactで書く場所

Reactは、画面、URL、共通部品、API通信、CSSを書く場所を分ける。
ページに全部書かず、役割ごとにファイルを分けると読みやすい。

最初に読み込む resources/js/app.jsx
React内のURL resources/js/routes
1ページ分の画面 resources/js/pages
共通部品 resources/js/components
API通信 resources/js/api
見た目 resources/css
