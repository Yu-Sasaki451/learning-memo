API用フォルダを作成

API通信の処理はページに直接書きすぎず、apiフォルダに分ける。


            mkdir -p resources/js/api

Todo API用の関数を作成

resources/js/api/todoApi.jsを書く。
Laravel API Resourceの返却は基本的にdataの中に入るので、画面側ではresponse.dataを見る。 


            export async function getTodos() {
                const response = await fetch('/api/todos', {
                    headers: {
                        Accept: 'application/json',
                    },
                });

                if (!response.ok) {
                    throw new Error('Todoの取得に失敗しました');
                }

                return response.json();
            }

            export async function createTodo(title) {
                const response = await fetch('/api/todos', {
                    method: 'POST',
                    headers: {
                        Accept: 'application/json',
                        'Content-Type': 'application/json',
                    },
                    body: JSON.stringify({
                        title: title,
                    }),
                });

                if (!response.ok) {
                    const error = await response.json();
                    throw new Error(error.message);
                }

                return response.json();
            }

画面でAPIを呼ぶ

resources/js/pages/Index.jsxを書く。
useEffectで初回表示時に一覧を取得し、フォーム送信で登録する。 


            import { useEffect, useState } from 'react';
            import { createTodo, getTodos } from '../api/todoApi';
            import '../../css/pages/Index.css';

            const Index = () => {
                const [todos, setTodos] = useState([]);
                const [title, setTitle] = useState('');
                const [isLoading, setIsLoading] = useState(true);
                const [errorMessage, setErrorMessage] = useState('');

                useEffect(() => {
                    let ignore = false;

                    async function loadTodos() {
                        try {
                            const response = await getTodos();

                            if (!ignore) {
                                setTodos(response.data);
                            }
                        } catch (error) {
                            if (!ignore) {
                                setErrorMessage(error.message);
                            }
                        } finally {
                            if (!ignore) {
                                setIsLoading(false);
                            }
                        }
                    }

                    loadTodos();

                    return () => {
                        ignore = true;
                    };
                }, []);

                const handleSubmit = async (e) => {
                    e.preventDefault();

                    if (title.trim() === '') {
                        setErrorMessage('Todoを入力してください');
                        return;
                    }

                    try {
                        const response = await createTodo(title);

                        setTodos([response.data, ...todos]);
                        setTitle('');
                        setErrorMessage('');
                    } catch (error) {
                        setErrorMessage(error.message);
                    }
                };

                return (
                    <main className="indexPage">
                        <h1>Todo</h1>

                        <form onSubmit={handleSubmit}>
                            <input
                                type="text"
                                value={title}
                                onChange={(e) => setTitle(e.target.value)}
                                placeholder="Todoを入力"
                            />
                            <button type="submit">登録</button>
                        </form>

                        {errorMessage !== '' ? <p>{errorMessage}</p> : null}
                        {isLoading ? <p>読み込み中</p> : null}

                        <ul>
                            {todos.map((todo) => (
                                <li key={todo.id}>{todo.title}</li>
                            ))}
                        </ul>
                    </main>
                );
            };

            export default Index;

このコードの注意点

!ignoreは、ignoreがfalseのときだけ処理するという意味。
useEffectのreturnでignoreをtrueにして、古いAPI通信の結果が後から画面に反映されるのを防ぐ。
setTodos([response.data, ...todos]) の ...todos は、既存のtodosを新しい配列へ展開する書き方。

削除APIも使う場合

削除もapiファイルに関数として分ける。


            export async function deleteTodo(id) {
                const response = await fetch(`/api/todos/${id}`, {
                    method: 'DELETE',
                    headers: {
                        Accept: 'application/json',
                    },
                });

                if (!response.ok) {
                    throw new Error('Todoの削除に失敗しました');
                }
            }

React Routerのデータ取得

React Routerを本格的に使う場合、ルート単位でデータ取得をまとめる方法もある。
まずはuseEffectで理解して、次にloaderやclientLoaderを調べるとよい。 


            export async function clientLoader() {
                const response = await fetch('/api/todos');
                const todos = await response.json();

                return todos;
            }
