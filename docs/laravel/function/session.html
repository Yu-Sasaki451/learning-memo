一時的に状態を保持する機能のこと

フラッシュメッセージ

処理後のリダイレクト先で1回だけメッセージを表示したいときに使う。
成功メッセージや「ログインしてください」などで使える。

ログイン状態の保持

HTTPはリクエストごとに独立しているので、ログイン状態の保持にセッションを使う。
ログイン成功後はsession()->regenerate()でセッションを再生成する。

routes/web.php

Route::get('/login', [AuthController::class, 'showLogin'])->name('login');
Route::post('/login', [AuthController::class, 'login']);
Route::post('/logout', [AuthController::class, 'logout'])->middleware('auth');

コントローラ

public function showLogin()
{
    return view('auth.login');
}

public function login(Request $request)
{
    $credentials = $request->validate([
        'email' => ['required', 'email'],
        'password' => ['required'],
    ]);

    if (Auth::attempt($credentials)) {
        $request->session()->regenerate();

        return redirect()->intended('/')
            ->with('message', 'ログインしました');
    }

    return back()->withErrors([
        'email' => 'メールアドレスまたはパスワードが違います',
    ])->onlyInput('email');
}

public function logout(Request $request)
{
    Auth::logout();
    $request->session()->invalidate();
    $request->session()->regenerateToken();

    return redirect('/login')->with('message', 'ログアウトしました');
}

検索、絞り込み条件の保持

検索条件をセッションに保存すると、別ページへ移動しても条件を保持できる。
リセット用の処理も用意しておく。

routes/web.php

Route::get('/todos', [TodoController::class, 'index']);
Route::post('/todos/search', [TodoController::class, 'search']);
Route::post('/todos/search/clear', [TodoController::class, 'clearSearch']);

コントローラ

public function search(Request $request)
{
    session(['todos.keyword' => $request->keyword]);

    return redirect('/todos');
}

public function index()
{
    $keyword = session('todos.keyword');
    $query = \App\Models\Todo::query();

    if (!empty($keyword)) {
        $query->where('content', 'like', "%{$keyword}%");
    }

    $todos = $query->latest()->paginate(10);

    return view('todos.index', compact('todos', 'keyword'));
}

public function clearSearch()
{
    session()->forget('todos.keyword');

    return redirect('/todos');
}

Blade

<form method="post" action="/todos/search">
    @csrf
    <input type="text" name="keyword" value="{{ $keyword }}">
    <button type="submit">検索</button>
</form>

マルチステップ

入力画面、確認画面、完了画面のように複数ページを使う場合、途中の入力内容をセッションに保存する。
完了後は不要になったセッションを消す。

確認画面へ進むコントローラ

public function confirm(Request $request)
{
    $request->validate([
        'content' => 'required|string|max:20',
    ]);

    session(['form.content' => $request->content]);

    return view('confirm', [
        'content' => $request->content,
    ]);
}

確認画面のBlade

<p>{{ $content }}</p>

<form action="/complete" method="post">
    @csrf
    <button>確定</button>
</form>

<form action="/back" method="post">
    @csrf
    <button>戻る</button>
</form>

完了処理

public function complete()
{
    $content = session('form.content');

    if (!$content) {
        return redirect('/input');
    }

    Todo::create([
        'content' => $content,
    ]);

    session()->forget('form');

    return redirect('/')->with('message', '登録しました');
}

戻る処理

public function back()
{
    return redirect('/input');
}
