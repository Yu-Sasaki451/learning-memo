Laravel

Laravelのコードを見る前に

LaravelはPHPなので、JavaScriptとは記号が違う。
まずは $、->、::、=>、use の意味を見ると、コードが読みやすい。

$todo $ は変数 PHPでは変数の名前の前に$を書く。
$todo->update() -> は中の処理を呼ぶ 取得したTodoに対してupdateを実行する。
Route::get() :: はクラスから呼ぶ Routeのgetメソッドを呼ぶ。
['title' => 'required'] => は対応 titleにはrequiredを対応させる。

Laravelで書く場所

Laravelは、URL、API処理、DB操作、入力チェックを書く場所が分かれている。
迷ったらまず全体の流れを見る。

画面URL routes/web.php に書く。
API URL routes/api.php に書く。
処理 app/Http/Controllers に書く。
入力チェック app/Http/Requests に書く。
DB database/migrations と app/Models に書く。
