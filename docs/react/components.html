Reactルーティングへ

React コンポーネント作成

ここは、React Routerで画面表示できた後にやる作業。
Header、Button、StoreFormをコンポーネントとして作り、ページから使う。

公式で確認した考え方

Reactコンポーネントは、JSXを返すJavaScript関数。
コンポーネント名は大文字で始める。
再利用したいUIは別ファイルに分け、exportして、使う側でimportする。
親から子へ渡す値はpropsとして受け取る。

<Header /> 部品を表示する形 Headerコンポーネントを画面に出す。
<Button type="submit">登録</Button> タグで囲む形 開始タグと終了タグの間に文字や子要素を入れる。
{children} {} はJSの値を使う場所 propsで受け取ったchildrenを表示している。
showSelect && (...) && は条件表示 showSelectがtrueの時だけ右側を表示する。

コンポーネント用フォルダを作成

componentsには再利用するUI、css/componentsにはそのCSSを置く。


            mkdir -p resources/js/components resources/css/components

Headerコンポーネントを作成

resources/js/components/Header.jsxを書く。共通のヘッダーはコンポーネントにする。


            import { Link } from 'react-router';
            import { ROUTES } from '../routes/routes';
            import '../../css/components/Header.css';

            const Header = () => {
                return (
                    <header className="header">
                        <Link className="headerLink" to={ROUTES.HOME}>
                            Home
                        </Link>
                        <Link className="headerLink" to={ROUTES.CATEGORY}>
                            Category
                        </Link>
                    </header>
                );
            };

            export default Header;

HeaderのCSSを作成

resources/css/components/Header.cssを書く。


            .header {
                display: flex;
                gap: 16px;
                padding: 16px 24px;
                border-bottom: 1px solid #dddddd;
            }

            .headerLink {
                color: #222222;
                font-weight: 700;
                text-decoration: none;
            }

Buttonコンポーネントを作成

resources/js/components/Button.jsxを書く。
childrenは、ボタンタグの中に書いた文字や要素を受け取るprops。 


            import '../../css/components/Button.css';

            const Button = ({ children, type = 'button', onClick }) => {
                return (
                    <button className="button" type={type} onClick={onClick}>
                        {children}
                    </button>
                );
            };

            export default Button;

ButtonのCSSを作成

resources/css/components/Button.cssを書く。


            .button {
                min-height: 40px;
                padding: 0 16px;
                border: 0;
                border-radius: 6px;
                color: #ffffff;
                font-weight: 700;
                background-color: #222222;
                cursor: pointer;
            }

登録フォームコンポーネントを作成

resources/js/components/StoreForm.jsxを書く。
onSubmit、placeholder、showSelectは親から渡すprops。
&&は左がtrueの時だけ右を表示するという意味。 


            import Button from './Button';
            import '../../css/components/StoreForm.css';

            const StoreForm = ({ onSubmit, placeholder, showSelect }) => {
                return (
                    <form className="storeForm" onSubmit={onSubmit}>
                        <input
                            className="formInput"
                            type="text"
                            placeholder={placeholder}
                        />

                        {showSelect && (
                            <select className="formSelect">
                                <option value="">選んでね</option>
                            </select>
                        )}

                        <Button type="submit">登録</Button>
                    </form>
                );
            };

            export default StoreForm;

登録フォームのCSSを作成

resources/css/components/StoreForm.cssを書く。ボタンの見た目はButton.css側で管理する。


            .storeForm {
                display: flex;
                gap: 12px;
                align-items: center;
                margin-top: 24px;
            }

            .formInput,
            .formSelect {
                min-height: 40px;
                padding: 0 12px;
                border: 1px solid #cccccc;
                border-radius: 6px;
            }

Indexページでコンポーネントを使う

resources/js/pages/Index.jsxを書き換える。


            import Header from '../components/Header';
            import StoreForm from '../components/StoreForm';
            import '../../css/pages/Index.css';

            const Index = () => {
                const handleSubmit = (e) => {
                    e.preventDefault();
                };

                return (
                    <>
                        <Header />
                        <main className="indexPage">
                            <h1>Todo</h1>
                            <StoreForm
                                onSubmit={handleSubmit}
                                placeholder="Todoを入力"
                                showSelect={false}
                            />
                        </main>
                    </>
                );
            };

            export default Index;
