自分で開発環境構築④

phpコンテナに入る

以降のphp artisanコマンドは、Laravelが入っているphpコンテナ内で実行する。


            docker compose exec php bash

シーダーファイル作成

まずシーディングは必須ではないのが前提。必要な時だけ作る。


            php artisan make:seeder SongSeeder

シーダーの中身を書く

database/seeders/SongSeeder.phpに書く例。


            <?php

            namespace Database\Seeders;

            use Illuminate\Database\Seeder;
            use Illuminate\Support\Facades\DB;

            class SongSeeder extends Seeder
            {
                public function run(): void
                {
                    DB::table('songs')->insert([
                        ['name' => 'Song A'],
                        ['name' => 'Song B'],
                    ]);
                }
            }

DatabaseSeederに登録

database/seeders/DatabaseSeeder.phpのrunメソッド内に書く。


            public function run(): void
            {
                $this->call(SongSeeder::class);
            }

シーディング実行


            php artisan db:seed
