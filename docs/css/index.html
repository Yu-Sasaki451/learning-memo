CSS

CSSはHTMLに見た目を付けるための言語。
最初は「どのHTMLに」「何を」「どんな値で」指定しているかを見る。

公式で確認したこと

初学では、まずセレクタ、プロパティ、値、波かっこの役割を分けて読む。

CSSはどこに何を書くか

HTML側にはclass名を書く。CSS側では、そのclass名の前に . を付けて対象を選ぶ。
その中に、色、文字サイズ、余白などの見た目を書く。

class="title" HTML側の目印 このHTMLにCSSを当てたい、という名前を付ける。
.title CSS側のセレクタ class="title" が付いたHTMLを選ぶ。
{ ... } 指定を書く場所 この中に、変えたい見た目をまとめて書く。
color: red; プロパティと値 colorが何を変えるか、redがどう変えるか。

CSSの基本形

セレクタは「どれに当てるか」。
プロパティは「何を変えるか」。
値は「どう変えるか」。 


            .title {
                color: #0b7285;
                font-size: 24px;
            }
.title class="title" が付いたHTMLを選ぶ。. はclassを選ぶ記号。
{ } この中に、titleに当てたい見た目をまとめる。
color: #0b7285; 文字色を指定している。colorが項目、#0b7285が色の値。
font-size: 24px; 文字サイズを指定している。pxは画面上の大きさの単位。

HTMLとCSSのつながり

HTMLとCSSは別ファイルでも、class名でつながる。
HTMLのclass名と、CSSの . の後ろの名前が同じなら、そのCSSが当たる。

HTMLに書く


            <h1 class="title">Todoアプリ</h1>
            <p class="lead">今日やることを管理する</p>

CSSに書く


            .title {
                color: #102a43;
            }

            .lead {
                color: #52606d;
            }
class="title" HTMLにtitleという目印を付けている。
.title CSSでtitleという目印のHTMLを選んでいる。
class="lead" 別のHTMLにはleadという別の目印を付けている。
.lead leadだけに別の見た目を当てている。

よく使うセレクタ

最初はclassセレクタを中心に使うと分かりやすい。
idは1ページに1つだけの場所、タグ名は同じタグ全部に当てたい時に使う。

.button class="button" が付いたHTMLを選ぶ。部品の見た目を作る時によく使う。
#app id="app" が付いたHTMLを選ぶ。Reactの表示場所など、1つだけの場所に使う。
h1 ページ内のh1タグ全部を選ぶ。広く当てたい時だけ使う。
.card .title cardの中にあるtitleだけを選ぶ。場所をしぼってCSSを当てたい時に使う。

余白はmarginとpaddingを分けて考える

marginは外側の余白。paddingは内側の余白。
カード同士を離したいならmargin、カードの中身をゆったりさせたいならpaddingを使う。 


            .card {
                margin-bottom: 16px;
                padding: 20px;
                border: 1px solid #d8e0e7;
            }
margin-bottom: 16px; cardの外側、下方向に16pxの余白を作る。
padding: 20px; cardの内側に20pxの余白を作る。
border: 1px solid #d8e0e7; 太さ1px、線の種類solid、色#d8e0e7の枠線を付ける。

横並びはflexから覚える

横並びはdisplay: flexから始める。
gapで要素同士の間隔を空け、align-itemsで上下の位置をそろえる。 


            .header {
                display: flex;
                align-items: center;
                gap: 12px;
            }
display: flex; headerの中身を横並びで扱う。
align-items: center; 上下方向の位置を中央にそろえる。
gap: 12px; 中の要素同士の間隔を12px空ける。
