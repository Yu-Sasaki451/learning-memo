シーディングへ

自分で開発環境構築③

phpコンテナに入る


            docker compose exec php bash

Laravelインストール


                composer create-project "laravel/laravel"=8.* . --prefer-dist

タイムゾーン変更

config/app.phpのtimezoneをAsia/Tokyoに変更する。


            'timezone' => 'Asia/Tokyo',

.envファイルのDB変更

docker-compose.ymlのMySQL設定に合わせて.envを変更する。


                DB_CONNECTION=mysql
                DB_HOST=mysql
                DB_PORT=3306
                DB_DATABASE=song_db
                DB_USERNAME=yorushika
                DB_PASSWORD=atogaki

コンフィグクリア


            php artisan config:clear

キャッシュクリア


                php artisan cache:clear

マイグレーションファイル(複数形)作成

songsテーブルを作る例。


            php artisan make:migration create_songs_table

テーブルにカラム追加

マイグレーションファイルのupメソッド内にカラムを追加する。


                $table->id();
                $table->string('name', 30)->nullable();

マイグレーション実行


            php artisan migrate

phpmyadminへ


                http://localhost:8080/
